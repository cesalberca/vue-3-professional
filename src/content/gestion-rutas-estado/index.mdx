## 📖 Introducción

Bienvenido al módulo **Gestión de rutas y estado**. En este módulo aprenderás a manejar dos aspectos fundamentales de las aplicaciones Vue modernas: el sistema de routing con Vue Router 4 y la gestión de estado global con Pinia, entendiendo las principales diferencias y mejoras respecto a sus versiones anteriores.

## 🎯 Objetivos del módulo

Al completar este módulo serás capaz de:

- **Migrar de Vue Router 3 a Vue Router 4** entendiendo todos los cambios fundamentales
- **Configurar y utilizar Vue Router 4** con las nuevas APIs y funcionalidades
- **Implementar gestión de estado** con Pinia como reemplazo moderno de Vuex
- **Aplicar mejores prácticas** en routing y gestión de estado en aplicaciones Vue 3

## 📚 Contenido del módulo

### Sección 1: Gestión de rutas con Vue Router 4

En esta sección nos centraremos en las **diferencias clave entre Vue Router v3 y v4**:

- **Inicialización**: Nuevas APIs de creación de router y aplicación
- **Cambios en la opción base**: Nueva forma de configurar la URL base
- **History**: Nuevos modos de historial y configuración
- **Opción fallback eliminada**: Cambios en el comportamiento de fallback
- **Rutas catch-all (\*)**: Eliminación de la sintaxis asterisco
- **currentRoute como ref()**: Nueva reactividad en la ruta actual
- **isReady() vs onReady**: Nueva API para detectar cuando el router está listo
- **scrollBehavior**: Cambios en el comportamiento de scroll
- **router-view, keep-alive y transition**: Nuevas formas de trabajar con estos componentes

### Sección 2: Gestión de estado con Pinia

En esta sección cubriremos:

- **De Vuex a Pinia**: Principales diferencias y ventajas
- **Stores con Composition API**: Nueva forma de definir stores
- **Estado reactivo y acciones**: Manejo moderno del estado
- **Getters y computed properties**: Propiedades derivadas optimizadas
- **TypeScript**: Soporte nativo y tipado seguro
- **DevTools y debugging**: Herramientas de desarrollo mejoradas
